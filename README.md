🎬 Netflix Clone

A responsive Netflix Clone built using HTML, CSS. This project replicates the UI of Netflix homepage with modern design and clean layout.

---

🚀 Features

- 🎥 Netflix-like homepage UI
- 📱 Fully responsive design
- 🎨 Clean and modern layout
- 🖼️ Background images and sections like real Netflix
- ⚡ Fast loading static website

---

🛠️ Tech Stack

- HTML5
- CSS3
- Google Fonts
